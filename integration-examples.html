🖨️ أمثلة دمج نظام الطباعة المتقدم

✅ هذا الملف يحتوي على أمثلة عملية لكيفية دمج نظام الطباعة المتقدم مع dashboard.html

1️⃣ الخطوة الأولى: إضافة المكتبة

أضف هذا السطر في head من dashboard.html:
<!-- مكتبة الطباعة المتقدمة --> <script src="test-printer.js"></script>
تأكد من أن ملف `test-printer.js` موجود في نفس مجلد `dashboard.html`

2️⃣ الطريقة الأولى: زر طباعة بسيط

إضافة زر في HTML:
<button onclick="printTestResultsFromAdmin('grade-id', 'test-id')" class="btn-print"> 🖨️ طباعة نتائج الاختبار </button>

3️⃣ الطريقة الثانية: إرسال بيانات مخصصة

استخدم هذا الكود عند جلب البيانات من API:
// جلب البيانات من API fetch('/api/test-results/123') .then(res => res.json()) .then(data => { // تنظيم البيانات const testData = { testName: data.test.name, testDate: data.test.date, testGrade: data.test.grade, testDuration: data.test.duration, passedStudents: data.passed, absentStudents: data.absent }; // فتح نافذة الطباعة testResultsPrinter.openPrintWindow(testData); });

4️⃣ الطريقة الثالثة: من جدول النتائج الموجود

إذا كنت لديك جدول نتائج موجود:
// جلب البيانات من الجدول الموجود function printFromExistingTable() { const testData = { testName: document.querySelector('[data-test-name]').textContent, testDate: new Date().toLocaleDateString('ar-EG'), testGrade: document.querySelector('[data-grade-name]').textContent, testDuration: '60 دقيقة', passedStudents: getPassedStudentsFromTable(), absentStudents: getAbsentStudentsFromTable() }; testResultsPrinter.openPrintWindow(testData); }
الدوال `getPassedStudentsFromTable` و `getAbsentStudentsFromTable` موجودة في `test-printer.js`

5️⃣ بنية البيانات المطلوبة

📊 بيانات الطالب الناجح:
{ rank: 1, // الترتيب (رقم) name: "أحمد محمود", // اسم الطالب (نص) rollNo: "001", // رقم الجلسة (نص) score: 95, // الدرجة المحققة (رقم) total: 100, // الدرجة الكلية (رقم) percentage: 95, // النسبة المئوية (رقم) grade: "A" // التقدير A, B, C, D, F }
⛔ بيانات الطالب الغائب:
{ rank: 1, // الترتيب (رقم) name: "أسماء خالد", // اسم الطالب (نص) rollNo: "016", // رقم الجلسة (نص) reason: "غياب مرضي" // سبب الغياب (نص) }

6️⃣ الدوال المتاحة

اسم الدالة الوصف المثال
openPrintWindow() فتح نافذة الطباعة testResultsPrinter.openPrintWindow(data)
setPassedStudents() تعيين الطلاب الناجحين testResultsPrinter.setPassedStudents([])
setAbsentStudents() تعيين الطلاب الغائبين testResultsPrinter.setAbsentStudents([])
getSummary() الحصول على الملخص الإحصائي const summary = testResultsPrinter.getSummary()
exportResultsToExcel() تصدير إلى Excel exportResultsToExcel('grade', 'test')

7️⃣ مثال عملي كامل

كود متكامل لطباعة نتائج فصل:
// دالة شاملة لطباعة النتائج async function printGradeTestResults(gradeId, testId) { try { // 1. جلب البيانات const response = await fetch( \`/api/grades/\${gradeId}/tests/\${testId}/results\` ); if (!response.ok) throw new Error('خطأ في جلب البيانات'); const { test, results } = await response.json(); // 2. تنظيم البيانات - الناجحون const passedStudents = results .filter(r => r.status === 'passed') .map((r, i) => ({ rank: i + 1, name: r.studentName, rollNo: r.studentId, score: r.score, total: r.totalScore, percentage: r.percentage, grade: getGradeFromPercentage(r.percentage) })); // 3. تنظيم البيانات - الغائبون const absentStudents = results .filter(r => r.status === 'absent') .map((r, i) => ({ rank: i + 1, name: r.studentName, rollNo: r.studentId, reason: r.absentReason || 'غياب' })); // 4. إعداد كائن البيانات الكامل const testData = { testName: test.name, testDate: new Date(test.date).toLocaleDateString('ar-EG'), testGrade: test.gradeName, testDuration: test.duration, passedStudents: passedStudents, absentStudents: absentStudents }; // 5. فتح نافذة الطباعة testResultsPrinter.openPrintWindow(testData); // 6. عرض رسالة نجاح alert('تم فتح نافذة الطباعة بنجاح!'); } catch (error) { console.error('خطأ:', error); alert('حدث خطأ: ' + error.message); } } // تحديد التقدير بناء على النسبة function getGradeFromPercentage(percentage) { if (percentage >= 90) return 'A'; if (percentage >= 80) return 'B'; if (percentage >= 70) return 'C'; if (percentage >= 60) return 'D'; return 'F'; }

8️⃣ خطوات التكامل مع dashboard.html

  1. انسخ الملفات الثلاثة إلى مجلد مشروعك:
    • ✅ test-printer.js
    • ✅ print-results.html
    • ✅ README.md (التوثيق)
  2. أضف المكتبة في head من dashboard.html: <script src="test-printer.js"></script>
  3. أضف زر الطباعة في جدول النتائج
  4. اختبر الوظيفة بالضغط على الزر
  5. اضبط البيانات حسب احتياجاتك
✅ انتهيت؟ الآن يمكنك الطباعة بضغطة واحدة!

9️⃣ نصائح وحيل

💡 نصائح مفيدة:
  1. استخدم F12 لفتح الـ Developer Tools واختبار الدوال
  2. تأكد من أن البيانات بصيغة JSON صحيحة
  3. استخدم console.log() لطباعة البيانات والتحقق منها
  4. جرّب window.print() الخاص بالمتصفح للمقارنة
  5. في الطباعة، استخدم معاينة الطباعة قبل الطباعة الفعلية
