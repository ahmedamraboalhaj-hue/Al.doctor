## 🚀 خطوات مطلوبة منك بعد رفع الملفات

### 1. ربط الموقع بـ Google Search Console (أهم خطوة)
1. اذهب إلى [search.google.com/search-console](https://search.google.com/search-console)
2. اضغط **إضافة موقع** → اختر **URL prefix** وأدخل:
   `https://aldoctor-7e153.web.app/`
3. اختر طريقة التحقق: **HTML tag** — ستظهر لك وسم مثل:
   ```html
   <meta name="google-site-verification" content="XXXXXXXXXXXXXXXX">
   ```
4. أضف هذا الوسم في `<head>` داخل `index.html` (قبل أي سكريبت)
5. اضغط **Verify** في Search Console
6. بعد التحقق → اذهب إلى **Sitemaps** → أضف:
   `https://aldoctor-7e153.web.app/sitemap.xml`

### 2. تحديث رابط الموقع في الملفات
> ⚠️ إذا كان رابط الموقع الفعلي مختلفاً عن `aldoctor-7e153.web.app`، استبدله في:
- كل وسوم `canonical` و`og:url` في الـ HTML
- ملف `sitemap.xml`
- ملف `robots.txt`

### 3. إنشاء صفحة "من نحن" (يُنصح بها جداً)
أنشئ صفحة `about.html` تحتوي على:
- سيرة ذاتية مختصرة للدكتور إبراهيم طناني
- خبراته في تدريس اللغة العربية
- منهجيته التعليمية
- صورته الشخصية (بـ alt="الدكتور إبراهيم طناني")

هذه الصفحة تساعد جوجل على بناء **Knowledge Panel** (البطاقة الجانبية) باسم الدكتور.

### 4. إضافة صورة Open Graph مخصصة
الصورة المثالية: **1200×630 بكسل** تحتوي على:
- صورة الدكتور أو شعار المنصة
- اسم "الدكتور إبراهيم طناني" بخط عريض
- سطر وصفي مثل "اللغة العربية للثانوية العامة والأزهرية"

ارفعها باسم `og-image.jpg` وحدّث رابطها في كل صفحة.

## 📊 تحسين سرعة الموقع (Core Web Vitals)

### مهم جداً لترتيب جوجل:

**أ. ضغط الصور**
- استخدم [squoosh.app](https://squoosh.app/) لتحويل الصور إلى WebP وتقليل حجمها
- أضف `loading="lazy"` لكل الصور الموجودة في الصفحة (ما عدا الصورة الأولى)

**ب. تحميل الخطوط بشكل أسرع**
في `index.html`، أضف هذا السطر مباشرة بعد `<meta charset>`:
```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
```
(موجود فعلاً — ✅ جيد)

**ج. تفعيل Gzip على Firebase Hosting**
أضف هذا في `firebase.json`:
```json
{
  "hosting": {
    "headers": [
      {
        "source": "**/*.@(js|css|html)",
        "headers": [{ "key": "Cache-Control", "value": "public, max-age=31536000" }]
      }
    ]
  }
}
```

## 🔗 بناء الروابط الخارجية (Link Building)

هذا العامل الأهم لكسب المراتب الأولى عند البحث عن "الدكتور إبراهيم طناني":

1. **إنشاء صفحة ويكيبيديا** (أو Wikiwand) باسم الدكتور — يُضاف رابط الموقع فيها
2. **Google Business Profile** — أنشئ ملف أعمال على خرائط جوجل (مجاناً) تحت اسم "الدكتور إبراهيم طناني"
3. **يوتيوب** — أضف رابط الموقع في وصف كل فيديو تعليمي
4. **واتساب وتيليغرام** — اطلب من الطلاب مشاركة رابط الموقع في المجموعات
5. **مواقع التعليم المصرية** — مثل `elmadrasa.com`، `esla7.com` — أضف اسمك ورابطك
